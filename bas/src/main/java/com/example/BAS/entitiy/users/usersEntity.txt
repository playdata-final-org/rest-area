1. LocalDate,LocalTime,LocalDateTime
   -3개의 클래스는 모두 타임존 개념을 갖고 있지 않기 때문에 타임존에 따른 시간 변환이 불가능
   -타임존: 동일한 로컬 시간을 따르는 지역

2.JsonIgnore
    -JSON 직렬화 역직렬화에서 속성을 무시하는데 사용 주로 민감한 정보나 중요한 정보를 노출되지 않게 하는 의도로 사용
    -불필요한 정보 필터링
    -순환참조방지
    -직렬화: 객체의 직렬화는 객체의 내용을 바이트 단위로 변환하여 파일 또는 네트워크를 통해서 스트림(송수신)이 가능하도록 하는 것을 의미
    -false or true로 비활성,활성
